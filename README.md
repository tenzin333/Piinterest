# Piinterest

Piinterest lets users view and upload images in a Pinterest-style layout
using React and Supabase.

## Features

-   User authentication
-   Upload and browse image pins
-   Mobile-friendly grid UI

## Install

    git clone https://github.com/tenzin333/Piinterest
    cd Piinterest
    npm install
    npm run dev

## Env

Create `.env`:

    VITE_APP_SUPABASE_URL=...
    VITE_APP_SUPABASE_KEY=...
